API Access

For advanced users, you can also access export data programmatically via our API:

# Export all user data
GET /api/export/user/data?format=json
# Export specific thread
GET /api/export/threads/{thread_id}?format=html&template=hugo
# Export site data
GET /api/export/sites/{site_id}?format=json

All export endpoints require authentication and only return your own data.
